
**一款剪切板，主要用于写文档或论文时，当做一个临时的复制粘贴的画板。
后续会继续更新，使之能够契合实际使用场景。**

************************************************
目前v1.1.2
功能：
1）支持最大200个粘贴项
2）支持鼠标处截图，提醒用户复制内容
3）支持文本和mathtype复制
4）支持开机自启
5）支持热键唤醒
6）支持窗口缩放
7）支持多种条目批量删除

使用方法：
1）无需安装，只需下载后，放在任意位置，打开test.exe程序
2）显示主界面，通过按下“Ctrl+alt+M”可以呼出界面，按下“Ctrl+alt+E”键隐藏主界面。

缺点：
1）目前还不能识别mathtype相同内容，因此，相同内容会被识别2次以上
2）不支持热键自定义
3）无法记录图片或者文件


持续更新中。。。
